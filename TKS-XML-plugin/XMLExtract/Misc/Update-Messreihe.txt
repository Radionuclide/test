Howto update class for tks xml plugin

Class name: Messreihe.cs
Location: TKS-XML-plugin\XMLExtract\Erzeugung

- Get latest xsd definition and its xsd version from TKS 
- Diff and merge with existing MessreiheReduziert.xsd 
    - Add only required Items
    - Add newly defined enum values esp. in EinheitEnum
    - Skip not used enums and types until requested explicitly
    
- call create-class.cmd to get updated Messreihe-xsd2code.cs
- copy Messreihe-xsd2code.cs to TKS-XML-plugin\XMLExtract\Erzeugung
- Diff and merge Messreihe-generated.cs with changes of Messreihe-xsd2code.cs 
    (most times Messreihe-generated.cs will be a copy of Messreihe-xsd2code.cs)

- Diff and merge new code of Messreihe-generated.cs into Messreihe.cs
- Update Messreihe-iba-mod.diff
